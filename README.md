# CoAura

## Core Definition
CoAura is the renderable, AI-first interface envelope through which a CoObject is exposed to minds and systems.

## Principle
One CoObject → many rendered CoAuras (domains, apps, surfaces)

## Layers
- CoObject (truth, external repo)
- CoAura (render + UX)
- CoStacks (runtime/operator)
- CoClouds (pressure/context)

Seed v0 — projection-safe, non-canonical
