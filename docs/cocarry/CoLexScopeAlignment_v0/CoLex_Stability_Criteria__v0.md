# CoLex Stability Criteria v0

## Success criteria
- terms are interoperable
- terms are non-duplicative
- terms are machine-resolvable across sessions
- seed set remains bounded
- candidate/adopted/deprecated states are distinguishable

## Expansion gates
A) CoIndex graph architecture
B) CoGibber encoding standard
C) CoVibe modulation system

These require explicit approval after stability is demonstrated.
