# CoLex Scope v0

## Mandate
CoLex is limited to:
- CoTerm lifecycle governance
- relationship mapping among CoLex, GIBindex, CoIndex, CoGibber, CoGibberTrue, CoVibe
- minimal schema definition
- seed index stabilization (<=15 terms)

## Explicit exclusions
- CoCloud expansion
- cultural substrate expansion
- n-dimensional ontology growth
- CoObject runtime layer
- full automation suites

## Rule
No expansion beyond language-layer unless stability criteria are met.
