# CoGuardian Heartbeat v0

## Purpose
Tray icon / visual micro-language for compressed system state.

## States
- Alive: calm pulse
- Idle: dim steady
- Searching: irregular slow pulse
- Warning: amber flicker
- Critical: red stutter

## Inputs
- CoPolicy state
- pointer resolution success
- CoWavelet success
- connectivity / coherence state

## Rule
Iconography must reflect real state, not cosmetic animation only.
