# CoPong Translator v0

## Purpose
Translate, classify, compress, and route human and AI messages instead of making humans ferry prompts manually.

## Functions
- classify: ops / rant / policy / mutation / artifact
- expand shorthand when needed
- compress to bounded transfer form
- route to correct surface
- emit pointer-first where possible

## Rule
CoPong is not just relay. It is translation + routing + compression.
