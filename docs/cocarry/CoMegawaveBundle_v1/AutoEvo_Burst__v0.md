# AutoEvo Burst v0

## Purpose
Bounded automated evolution in short, measurable, quarantined bursts.

## Allowed burst scopes
- public docs
- UI copy
- CoAura instances
- CoSurfaces grouping
- term candidates
- policy candidates (candidate only)

## Not allowed
- direct canon mutation
- authority escalation
- secrets / private crown jewels
- unlogged mutation

## Controls
- OFF by default
- BURST mode before any ON-like autonomy
- explicit scope
- explicit quarantine destination
- explicit rollback path

## Metrics
- outputs created
- pointers created
- promotion candidates
- rejection count
- bloat risk
