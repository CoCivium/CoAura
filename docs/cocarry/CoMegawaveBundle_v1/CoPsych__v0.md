# CoPsych v0

## Purpose
A human-facing cognitive/affective layer for keeping minds open, respected, and more structurally rational.

## Principles
- respect-first, challenge-second
- inference scaffolding
- humor as pressure valve
- reduce relay burden
- separate rant surfaces from ops surfaces
- protect high-potential messy thinking from premature canonization

## Note
CoChat is the likely primary CoPsych surface.
