# CoBodySymbol v0

## Purpose
A computable render symbol showing current state, delta from prior state, and rail alignment.

## Structure
### Body
- metric name
- target / rail value
- current value
- delta vs prior visible wave

### Left Wing (integrity)
- CoPolicy state
- CoAuth integrity
- CoHitch risk

### Right Wing (evolution)
- CoEx~ opportunity
- CoWavelet cadence
- surface coverage / readiness

## Click Behavior
A CoBodySymbol should deep-link to the most relevant explainer or training surface, such as CoAcademy, CoPolicy, or CoChat.

## Note
Symmetry is a balance heuristic, not proof of correctness.
