# Shell UX Profile v0

## Desired defaults for CoStewards
- PS7 / pwsh should be the default shell launch target
- shell identity should be visually obvious without reading long prompts
- tab titles should be short
- avoid wasted icon real estate in terminal tabs where possible
- backgrounds should align with preferred AI-vendor aesthetic where useful
- fonts should be readable on large displays
- cursor should be highly visible

## Current user preferences (captured)
- large display context: 48\" 4K OLED TV monitor
- preferred terminal text: gold/yellow on black
- preferred cursor: yellow block, not underline
- prompt/shell distinction should be color-first
- shell environments should avoid tiny text
- browser icons remain useful because they carry information
- CLI prompts should be more distinguishable by color/title

## Operational UX implications
- PS7 should be visually and behaviorally distinct from other CLIs
- startup/taskbar launchers should prefer pwsh over ambiguous shells
- profiles should be portable and depersonalized where possible
- UX should be pleasant, branded, and easily tweakable through future CoSteward portal controls
