# InSeed Product Lines v0

## Direction
Subtle, low-pressure path for legacy-org leaders to explore AI-era pivots without obvious commercial aggression.

## Candidate lines
1. CoAudit seminar / debrief
2. executive AI-governance independent perspective
3. transition-office design for parallel suborg experimentation
4. org pivot potential analysis
5. post-hierarchy / meritocratic / ad-hoc org exploration
6. implementation support for gradual pivot toward AI-teamed operating models

## Framing
- no hard-sell
- no crowns
- executives participate as peers during some seminar modes
- ties into CoAcademy / EntMent softly
