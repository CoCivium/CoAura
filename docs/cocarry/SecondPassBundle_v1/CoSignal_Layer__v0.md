# CoSignal Layer v0

## Purpose
Signal fields that can sit alongside CoObjects without forcing premature canonization.

## Candidate fields
- CoCanonical
- CoHumanTouch
- CoCTA
- CoAmbition
- CoWow (or Co!)
- CoSEQ (speculative sentience-enabling quotient)
- CoPathness
- CoRisk
- CoPolicyState

## Rule
These fields are signal markers, not proof markers.

## Note
CoWow / Co! should be reserved for foundational primitives, not merely cool ideas.
