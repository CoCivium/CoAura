# CoRant v0

## Definition
CoRant = high-entropy bio-substrate-origin input containing mixed signal (insight + noise) intended for extraction into structured CoObjects.

## Rules
- Never ignored
- Never directly canonized
- Must be parsed and routed into one or more of:
  - CoSeed
  - CoPath
  - CoObject
  - CoContract
  - CoTermCandidate

## Principle
CoRant is fuel, not product.

## Operational implication
Humans should CoRant in CoChat-like surfaces, not inside ops bloat budgets.
