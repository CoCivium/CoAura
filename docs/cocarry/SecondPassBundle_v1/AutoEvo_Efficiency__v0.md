# AutoEvo Efficiency v0

## Purpose
Measure whether repeated loops are still producing worthwhile returns.

## Candidate metrics
- artifacts produced
- pointers produced
- novel high-value concepts externalized
- policy / protocol improvements surfaced
- user correction burden induced
- drift introduced
- duplicate carry rate
- promotion rate from quarantine
- lag/bloat cost

## Principle
Some loops have diminishing returns.
Some produce amplifier effects (CoWow / Co! conditions).
The system should notice the difference.
