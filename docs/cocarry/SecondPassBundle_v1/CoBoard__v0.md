# CoBoard v0

## Definition
CoBoard = temporal, local-first context substrate for copied/selected micro-artifacts.

## Lifecycle
- Fresh
- Fading
- Dormant
- Gone

## Principle
Nothing is permanently off; everything fades unless kept.

## Core actions
- Keep
- Share
- Route
- Clear / Release

## Rule
Selection stays; beautification decays.
Do not mutate actual text readability to simulate decay.
Use a life-layer / aura-layer instead.
