# Dashboard Derivation Rule v0

## Rule
The CoSteward dashboard is a derived adjacency layer over canonical bootstrap artifacts.

## Canonical truth
- CoGoAll / CoBeacon
- CoBus
- RAW pointers

## Non-canonical adjacency
- CoSteward dashboard
- CoGuardian UI
- CoSurfaces rendering
- helper navigation

## Hard rule
Adjacency must never become required for operation.
