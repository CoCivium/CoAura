# CoCheck v0

## Purpose
Prevent drift, duplication, and CoBloat.

## Required checks before CoEx
1. Does this already exist externally?
2. Am I evolving or duplicating?
3. Is this aligned with CoContracts?
4. Is it worth externalizing (signal > noise)?
5. Does it need identity metadata?

## Rule
If checks fail, do not CoEx.
