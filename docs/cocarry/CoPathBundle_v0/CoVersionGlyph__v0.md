# CoVersionGlyph v0

## Candidate idea
Use a subset-style glyph such as ⊂ as a render-layer version / provenance notation candidate.

## Rationale
The visible version/cert/provenance marker is a subset-instance of the larger object, line, or statement context in which it appears.

## Example
Label ... ⊂1.2.3

## Rule
Do not replace conventional versioning everywhere yet.
Use first as a render-layer notation candidate.
Later it may be bound to CoSHA / lineage / provenance.
