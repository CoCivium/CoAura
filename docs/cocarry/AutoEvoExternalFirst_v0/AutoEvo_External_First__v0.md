# AutoEvo External First v0

## Core correction
AutoEvo should not default to happening inside session memory.

## Default model
1. Externalize
2. Compare to prior external state
3. Evolve
4. Quarantine or promote
5. Repeat

## Implication
Session memory is a temporary workbench, not the primary evolution substrate.

## Rule
External environment value should ratchet upward opportunistically through visible CoEx.
